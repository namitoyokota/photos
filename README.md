# photos.namitoyokota.com

-   **Framework**: [Angular](https://angular.io/)
-   **Deployment**: [Vercel](https://vercel.com)

## Running Locally

```shell
$ git clone https://github.com/namitoyokota/photos.git
$ cd photos
$ yarn install
$ yarn dev
```
